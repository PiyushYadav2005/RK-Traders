# RK Traders Frontend

A modern, responsive electrical products website built with React, TypeScript, Vite, and Tailwind CSS.

## Features

- Responsive UI
- Product Catalog
- Product Details
- Shopping Cart
- Checkout
- Contact Page
- Gallery
- Quote Request
- Admin Login
- Category Browsing
- Brand Showcase

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router

## Installation

```bash
npm install
npm run dev
```
